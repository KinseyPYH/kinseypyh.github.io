Projects

badminton-scores.com

This website displays live professional badminton scores. Users are able to follow professional badminton matches as they are played. Users are also able to click on the calendar on the top left to choose a previous match day and display all match results for that day. Click the photo to head to the website!

Backend

The backend is hosted on AWS EC2 and uses Node.js. It has a couple of functions:
1. It performs API queries to collect scores
2. Sends these scores to the front end
3. Stores the scores into a database for use

Frontend

The frontend simply uses HTML, CSS, and JS. To collect match data, it uses AJAX to make HTTP calls to the backend server.

Database

The database used to store match data is MongoDB Atlas. The match data is currently stored by date.

Semi-Autonomous Path-Finding Robot

This robot is a path-finding robot. Its function is to drive through the warehouse and scan all inventory. My contribution was to enable semi-autonomous driving. This project was successful in that it was able to, given a pre-mapped space, travel from point A to point B, avoiding obstacles in the way.

Robot Framework

The robot used Robot Operating System (ROS) to control individual motors and sensors. The robot had a Jetson Nano as its onboard computer.

Path Planning

The path planning utilised a LIDAR sensor, an IMU, and SLAM.

The data from the LIDAR and the IMU were combined using the Extended Kalman Filter and were able to be used in the SLAM library for path finding.
